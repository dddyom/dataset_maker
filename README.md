# dataset_maker

Скрипт для создания обучающих и проверяющих сборок для нейросетей. Подматрицы с целями и помехами получаются из буферов формата txt, содержащих в себе матрицы - снимки с ИКО. 

### Для работы

Необходимо заполнить config путями (в комментариях указано, какими). Либо можно оставить все пути пустыми, в таком случае, необходимо будет заполнять их во время работы, через консоль.

После заполнения конфига, нужно запустить файл main. Его посредством происходит управление данными. 

### Функционал

Можно добавить в базу данных матрицу, преборазовав её в numpy массив.

Получить и добавить в базу данных связку снимков с заданными размерами, вырезанными из матрицы. По координатам. 

Создать сборку train или test из уже существующих значений в базе данных, либо из новых.

Склеить сборки train и test для подачи полной сборки на вход нейросети.

Преобразовать train или test сборки в png для контроля значений и очистки.
